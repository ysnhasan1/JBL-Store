## About The Project
In this project, I wanted to create an E-Commerce website using **HTML**, **SCSS** and **Bootstrap**. <br>
I focused only on the design because I did not use JavaScript in the project. Therefore, there are no functional features in the project. <br>
There are three different categories of products on the website, and each category consists of three products. <br>
Each product has a detail page. <br>
In addition, there are About and Frequently Asked Questions (FAQ) pages. <br><br>
